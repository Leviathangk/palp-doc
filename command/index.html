Command

命令行工具

cmd 中输入 palp 即可看到帮助信息

palp 版本号

Palp 操作命令如下
        palp create [options] [args]

可选 options:
        -p              即 project，创建爬虫项目
        -s              即 spider，创建爬虫

可选 args:
        1               创建 spider 时，创建普通 spider
        2               创建 spider 时，创建分布式 spider
        3               创建 spider 时，创建周期 spider

示例:
        palp create -p demo
        palp create -s baidu 1

创建项目

创建项目和爬虫都会去判断文件是否存在

palp create -p xxx

创建爬虫

创建爬虫时，会自动移动到 spiders 文件夹下（查找为 3 个深度）

palp create -s xxx 1
